# Модуль jiralite-service

Модуль `jiralite-service` содержит сервисные классы, реализующие бизнес-логику для системы управления задачами Jira Lite.

## Структура

### Основные сервисы:

-   `AuthService` - сервис авторизации, аутентификации и управления пользователями
-   `BoardService` - сервис для управления досками и связанными с ними проектами
-   `IssueService` - сервис для управления задачами и их статусами

## Архитектура

Сервисный слой реализует паттерн Singleton для большинства сервисов, обеспечивая единую точку доступа к бизнес-логике.

Взаимодействие с данными осуществляется через доменные объекты из модуля `jiralite-domain`.

## Зависимости

Модуль зависит от:

-   `jiralite-domain` - доменные объекты, с которыми работают сервисы
-   Библиотек логирования (SLF4J + Logback) для ведения журнала операций

## Использование

Сервисы используются в UI-слое для:

-   Авторизации пользователей
-   Отображения списка проектов и досок
-   Управления задачами (создание, редактирование, перемещение)
-   Работы с Kanban-доской
