# Модуль jiralite-domain

Модуль `jiralite-domain` содержит доменные классы для системы управления задачами Jira Lite.

## Структура

### Основные доменные классы:

-   `User` - пользователь системы
-   `Project` - проект, содержащий доски и задачи
-   `Board` - канбан-доска с колонками для отображения задач
-   `Issue` - задача с описанием, статусом и приоритетом
-   `Comment` - комментарий к задаче

### Перечисления:

-   `Status` - статус задачи (TO_DO, IN_PROGRESS, DONE)
-   `Priority` - приоритет задачи (LOW, MEDIUM, HIGH)
-   `Role` - роль пользователя (USER, ADMIN)

## Зависимости

Модуль использует JPA и Hibernate для объектно-реляционного отображения. Доменные классы размечены аннотациями JPA для корректного маппинга в базу данных.

## Использование

Доменные классы используются в других модулях системы:

-   В `jiralite-service` для реализации бизнес-логики
-   В `jiralite-ui` для отображения данных в пользовательском интерфейсе
